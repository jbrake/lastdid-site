--- layout: help title: "Folders" summary: "Group related tasks together" description: "Group related tasks together in LastDid using folders." ---

Folders let you group related tasks together — "Health", "Home", "Friends & Family", whatever makes sense for you. They're optional. Plenty of people use LastDid happily without ever creating one.

Creating a folder

Long-press any task → Move to Folder → New Folder. Give it a name and a color. The task you long-pressed will be the first one inside.

Navigating in and out

Tap a folder card to enter it. While inside, you'll see only the tasks in that folder, and the + button creates new tasks directly inside that folder. Use the back arrow in the top left to return to the top level.

Renaming, deleting, moving things around

  • Long-press a folder card to rename it
  • Long-press a folder card → Delete Folder to remove it (tasks inside aren't deleted, just moved out)
  • Long-press a task → Move to Folder (or Change Folder, if it's already in one) to put it somewhere else
  • Long-press a task that's currently in a folder → Remove from Folder to take it back to the top level
