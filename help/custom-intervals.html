--- layout: help title: "Custom intervals" summary: "A gentle glow for tasks you want to repeat on a cadence" description: "Set a gentle visual cadence reminder for repeating tasks in LastDid." ---

Some activities you want to do on a rough cadence — call your parents every couple of weeks, water plants every few days. LastDid lets you set a custom interval per task as a gentle visual cue.

Setting an interval

Long-press the task → Edit → Activity Interval → enter how many days. As that interval approaches and passes, the task tile gradually glows a little brighter to make it easier to notice.

It's not a deadline

This is intentionally gentle. The tile glows; it doesn't turn red, it doesn't notify you, it doesn't say "overdue". It's a visual nudge, not a scolding. If you blow past the interval, that's fine — the glow just sticks around until you log it.

Removing an interval

To clear it, edit the task and leave the Activity Interval field blank. The tile will go back to behaving like every other task.
