Sorting

Tap the ellipsis (•••) icon in the top right of the Tasks tab to change the sort order. Four options:

  • Most Recent — tasks you logged most recently float to the top
  • A-Z — alphabetical by task name
  • Z-A — reverse alphabetical
  • Manual — drag tiles to put them in any order you want

Grid vs list

The same menu lets you switch between grid view (compact tiles, two columns) and list view (one row per task with more visible at a glance, like the last log time and notes preview).

Searching

Tap the magnifying-glass icon to search by task name. This is useful when you have a lot of tasks. LastDid also indexes your tasks in iOS Spotlight, so you can find them by swiping down on your home screen and typing.
